# Porfolio Ramziatou AFFO TOSSOU

## Contexte
Cette SAE consiste en la création d’un site web personnel dans le cadre
du BUT Réseaux & Télécommunications.

Le site est réalisé individuellement à partir d’un template Bootstrap
et hébergé sur GitHub.

## Objectifs
- Créer un site web responsive
- Adapter un template Bootstrap
- Présenter son parcours et ses compétences
- Respecter les recommandations W3C

## Contenu du site
- Présentation personnelle
- CV en HTML/CSS
- Portfolio des compétences du PN R&T :
  - Administrer
  - Connecter
  - Programmer
  - Sécuriser
  - Surveiller


## Compétences développées
- Programmer
- Communiquer et se présenter professionnellement
